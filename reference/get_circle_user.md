# Get Circle CI user

Retrieve details about the authenticated Circle CI user.

## Usage

``` r
get_circle_user()
```

## Value

A named vector of class `circle_user` containing information about about
the authenticated user:

- Full name

- Username

- ID

- API endpoint

## Examples

``` r
if (FALSE) { # \dontrun{
get_circle_user()
} # }
```
